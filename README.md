## FYP Project

To pull from github: 
git pull origin main

To push to github: 
1. git status
2. git add .
3. git commit -m "Your message"
4. git push origin main

To start server: 
1. cd server
2. venv/Scripts/activate
3. python backend.py

To start frontend: 
1. cd frontend
2. npm start
